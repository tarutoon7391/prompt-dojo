GIGOを超えろ。

ダメなプロンプトからはダメなものしか生まれない。構造・役割・制約・出力形式を体に叩き込んで、AIから最大の出力を引き出せ。

CHEAT SHEET

プロンプト設計の鉄則を凝縮。いつでも見返せる一覧。

// 基本構造

ROLE
「〜として振る舞ってください」でAIに専門家を演じさせる
TASK
動詞から始める。書いてより〜を〜向けに書いて
CONTEXT
背景・目的・対象読者を添える
FORMAT
出力形式を明示。箇条書き 表形式 JSON
CONSTRAINT
文字数・禁止事項・トーンを数値で指定

// 精度を上げるテクニック

Few-shot
「例：〜」を添えて出力パターンを学習させる
CoT
ステップバイステップで考えてくださいで思考を展開
Self-check
出力後に改善点を3つ挙げてくださいで自己レビュー
Negative
〜は使わないでの否定制約で余計な出力を排除
Persona
ターゲット読者を具体的に設定。28歳・エンジニア・副業初心者

// GIGO対策チェックリスト

✓ 役割

✓ 目的
「何のために」が明確か？
✓ 対象
「誰向けに」が入っているか？
✓ 形式
出力フォーマットを指定しているか？
✓ 制約
文字数・禁止語・トーンを数値化しているか？
